# Agent chats (repo copy)

Cursor stores full chat transcripts on your machine, not inside this repository. To avoid **context gaps** when you work from another device or start a new chat, store what matters here.

## What to add

- **Summary (recommended):** After a long thread, paste a short markdown note: goal, key conclusions, file paths touched, and follow-ups.
- **Full export (optional):** Copy transcript content or attach an export file. Prefer redacting secrets. Watch repository size.

## Source folder (Windows)

`%USERPROFILE%\.cursor\projects\<workspace-folder>\agent-transcripts\`

The workspace folder name follows your checkout path. Use filenames like `2026-04-22-topic-summary.md` so history sorts chronologically.
